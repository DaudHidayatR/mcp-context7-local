Use this skill when Anthropic visual identity should guide the output.

Brand palette:
- Dark: `#141413`
- Light: `#faf9f5`
- Mid gray: `#b0aea5`
- Orange: `#d97757`
- Blue: `#6a9bcc`
- Green: `#788c5d`

Fonts:
- Headings: Poppins
- Body: Lora

Guidance:
- Keep brand usage consistent across the full artifact
- Do not mix in unrelated palette or font choices once this skill is active
