description: Co-author proposals, specs, PRDs, RFCs, and decision docs with an iterative structure-first process. Use for substantive written documents.
---

Use this skill for collaborative document development.

Three-stage workflow:
1. Context gathering
2. Refinement and structure
3. Reader testing

Guidance:
- Start by clarifying missing context
- Build the document section by section
- Test the draft from the perspective of a fresh reader
- Optimize for decision quality, not just polished prose
