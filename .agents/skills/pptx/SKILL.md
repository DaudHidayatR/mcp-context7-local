---
name: pptx
description: Create and edit PPTX slide decks with strong visual quality and a required QA loop. Use for presentations, pitch decks, and slide-based outputs.
---

Use this skill for any PowerPoint task.

Creation modes:
- From template: unpack and edit XML, then repack
- From scratch: use PptxGenJS

Design rules:
- Choose a topic-specific palette, not a generic corporate blue theme
- Every slide needs a visual element
- Vary layouts across the deck
- Avoid decorative accent lines under titles
- Keep title and body sizes clearly separated

QA loop:
1. Extract slide text
2. Render slides to images
3. Inspect visually
4. Fix and re-check
