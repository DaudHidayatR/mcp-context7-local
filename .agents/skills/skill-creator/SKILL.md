---
name: skill-creator
description: Create and improve skills through an evaluation loop. Use when defining a new skill, refining triggers, or comparing with-and-without skill behavior.
---

Use this skill to build or improve skills systematically.

Core loop:
1. Capture intent and write `SKILL.md`
2. Create test cases
3. Run with and without the skill
4. Grade outputs and refine

Skill anatomy:
- `SKILL.md` is required
- `scripts/`, `references/`, and `assets/` are optional

Guidance:
- Optimize the trigger description, not just the long body
- Use evaluation and comparison as part of the workflow
