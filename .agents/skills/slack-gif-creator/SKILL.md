Use this skill for Slack-targeted GIF generation.

Target sizes:
- Emoji: 128x128, short duration, Slack-safe frame rate
- Message GIF: 480x480

Animation patterns:
- Shake
- Pulse
- Bounce
- Spin
- Fade
- Slide
- Zoom
- Particle burst

Guidance:
- Keep animations short and legible
- Validate dimensions and playback constraints
- Prefer smooth easing over abrupt motion
