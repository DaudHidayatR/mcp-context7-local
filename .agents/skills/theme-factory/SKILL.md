---
name: theme-factory
description: Apply polished visual themes with coherent palettes and font pairings. Use when the user wants a themed artifact rather than raw functional output.
---

Use this skill to choose or generate a coherent theme for an artifact.

Reference themes include:
- Ocean Depths
- Sunset Boulevard
- Forest Canopy
- Modern Minimalist
- Golden Hour
- Arctic Frost
- Desert Rose
- Tech Innovation
- Botanical Garden
- Midnight Galaxy

Guidance:
- Choose one complete theme direction
- Keep color and typography internally consistent
- Match the theme to the artifact purpose, not just personal taste
