---
name: web-artifacts-builder
description: Build multi-component frontend artifacts with a React and TypeScript oriented workflow. Use for richer UI deliverables that need bundling and handoff.
---

Use this skill for complex frontend artifacts rather than small one-file UI tasks.

Preferred stack:
- React 18
- TypeScript
- Vite
- Parcel
- Tailwind CSS
- shadcn/ui

Workflow:
1. Initialize the artifact project
2. Edit generated code
3. Bundle the artifact to a shareable HTML output
4. Verify the final artifact works as a standalone deliverable
